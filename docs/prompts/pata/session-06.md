### Co máme k dispozici

**DB data:** `recipe_steps` + `recipe_step_translations` (instruction, tip, default_timer_seconds) — 77 kroků pro 20 receptů. `recipe_ingredients` + `ingredients` — 145 ingrediencí. Vše funguje s přímým DB připojením (RLS propouští server queries).

**IDs pro landing page:** Guláš = `10000000-...17`, Pad Thai = `10000000-...06`, Boeuf Bourguignon = `10000000-...03`

**Znovupoužitelné komponenty:** `LevelSummary`, `LevelCookingMode`, `LevelCompletion` — všechny fungují bez modifikací.

---

### Architektura

```
/[lang]/game/freeplay/[recipeId]/page.tsx  ← nová server stránka (data fetch)
  └── FreeplayRecipeExperience.tsx          ← nový client state machine
        ├── LevelSummary          (reuse)   ← přehled + tlačítko "Vařit"
        ├── LevelCookingMode      (reuse)   ← kroky, timer, AI chef
        └── LevelCompletion       (reuse)   ← hotovo, zpět na freeplay
```

### Co se vytvoří / upraví

| Soubor | Akce |
|---|---|
| `app/[lang]/game/freeplay/[recipeId]/page.tsx` | **Nový** — server component, fetchne recept+kroky+ingredience z DB, předá do experience |
| `app/components/game/freeplay/FreeplayRecipeExperience.tsx` | **Nový** — 3-fázový state machine: `summary → cooking → completion` |
| `app/components/RecipeGrid.tsx` | **Upravit** — `RecipeCard` → `<Link href="…/[id]">` |
| `app/[lang]/page.tsx` | **Upravit** — landing page karty dostanou konkrétní `href` na detail |
| `app/[lang]/dictionaries/cs.json` + `en.json` | **Upravit** — přidat label "Zpět na recepty", freeplay completion texty |
